Description

เลข Fibonacci ของ n จะมีค่าเท่ากับ ค่า Fibonacci ของ n-1 รวมกับ ค่า Fibonacci ของ n-2 เขียนเป็นสมการดังนี้

F(n) = F(n-1) + F(n-2)
โดยที่
F(0) = 0
F(1) = 1

เช่น
F(2) = F(1) + F(0) = 0 + 1 = 1
F(3) = F(2) + F(1) = 1 + 1 = 2

จงเขียนโปรแกรมหาค่า Fibonacci ของ n เช่น n = 10 จะมีค่า F(10) = 55
